# SolarSuite Web (Next.js)

## Requisiti
- Node.js 18+ (consigliato 20)

## Avvio in locale
```bash
npm install
npm run dev
```
Apri: http://localhost:3000

## Pubblicazione veloce (Vercel)
1. Carica questa cartella su GitHub
2. Importa il repo su Vercel e fai Deploy

## Contatti
La pagina /contact usa `mailto:`. Cambia la tua email in:
`app/contact/page.tsx` (cerca `mailto:tuoindirizzo@email.it`).
